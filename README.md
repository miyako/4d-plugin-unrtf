## Syntax

```
text:=UnRTF(path{;format})
```

Parameter|Type|Description
------------|------------|----
path|TEXT|``UnRTF Output HTML`` ``UnRTF Output plain text`` ``UnRTF Output VT100`` ``UnRTF Output LaTeX``
format|LONGINT|
text|TEXT|
